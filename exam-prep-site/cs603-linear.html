CS603 linear course path

CS603 全教材线性学习：从逻辑到 Dafny、模型检测和 SMT

这页按课程知识依赖顺序学习。先学逻辑和 Hoare triples，再学 Dafny 如何把证明写进程序，最后学模型检测、LTL/CTL、SAT/SMT 和需求形式化。

1. 课程全景：为什么要形式化验证

测试只能覆盖有限例子；形式化验证试图证明一类输入或一类行为全部满足性质。

顺序内容材料过关标准
1.1Module overview / failures1. ModuleOverview.pdf, Software-Fails-Watch.pdf, FormalMethodsinIndustry.pdf知道形式化方法解决什么问题。
1.2Design / Programming by ContractDesign by Contract.pdf, Programming by Contract.pdf能解释 precondition、postcondition、invariant。

2. Logic 和 Hoare Logic

先学如何把句子变成逻辑公式，再学如何把程序片段夹在前后条件之间证明。

顺序内容材料过关标准
2.1Predicate / propositional logic1_logic_603.pdf, labZ3_PropLogic.pdf能解释 predicate、tautology、satisfiable、unsatisfiable。
2.2Hoare logic2_hoarelogic_RM.pdf, Hoare Logic Summary Sheet.pdf, labs/exercises能写 partial correctness proof。
2.3LoopsExercises from Lecture_ Hoare Logic Loop.pdf能说明 invariant establishment/preservation/exit。

3. Dafny：程序和证明合在一起

Dafny 用 requires/ensures/invariant/decreases 告诉 verifier 该证明什么。

顺序内容材料过关标准
3.1Dafny intro1.Dafny_Intro.pdf, 2.Dafny.pdf能解释 requires、ensures、assert、ghost。
3.2Recursion and termination3.RecursionandTermination.pdf, Termination Exercise Sheet 2.pdf能解释 decreases 为什么证明终止。
3.3Loops and iterative programs4.DafnyLoops.pdf, 5.Iterative.pdf能找 invariant 和 variant。
3.4ArraysArrays I.pdf, 7.ArraysII_short.pdf, LinearSearch.pdf能写数组范围和 forall invariant。
3.5Objects/classes8.ObjectI2023.pdf能解释 class invariant、constructor contract、ghost predicate。

4. Requirements、Refinement 和 Event-B

在写代码前，需求本身也可能含糊。Formalising requirements 和 data refinement 讨论如何从自然语言需求走向更精确模型。

内容材料过关标准
Formalising requirements / FRETFormalisingRequirements.pdf, 2026_Slides_Formalise.pdf能解释 FRET 如何减少自然语言歧义。
Event-B / data refinementEventB-Summary-refcard.pdf, 2024_Slides_FMAS.pdf能解释 abstract model 到 concrete implementation。

5. Model Checking 和 Temporal Logic

Hoare/Dafny 适合程序前后条件；model checking 适合状态系统和时间性质。

顺序内容材料过关标准
5.1Kripke structuresMCLecture1CS603.pdf, MCLecture2CS603.pdf能解释 state、transition、label、path。
5.2LTL21-ltl_rm.pdf, ltl_vissoft.pdf, temporal-examples_2023.pdf能用 G/F/X/U 写性质。
5.3CTL22-ctl_rm.pdf, CTL_Exercises.pdf能区分 A/E 路径量词。
5.4Spin / NuSMVMCL41VerifyingWithSPIN_cs603.pdf, Verifying with Spin (TL).pdf, SpinTutorial_fromSpinroot.pdf能比较 Spin 和 NuSMV。

6. SAT、SMT、Z3 和 AI4SE

最后学自动推理工具。SAT 只处理布尔真假；SMT 还懂整数、数组等理论；Z3 是 SMT solver。

内容材料过关标准
SAT / DPLL / CDCL / CNFsld_mth1.pdf, labZ3_PropLogic.pdf能解释 sat/unsat/model 和 DPLL 基本思路。
SMT / Z3Z3Exercises.pdf, labZ3_PropLogic.pdf知道 Z3 如何用 check-sat/get-model。
AI4SE / neural network verificationAI4SE_Monahan1.pdf, CS603_FormalVerification_NeuralNetworks.pdf知道形式化方法也可用于 AI/software engineering。

7. 最后接到考试

读完线性页后，回到 CS603 题型页 按 Q1-Q4 练：logic/Hoare、Dafny、FMAS/SAT、model checking。
