CS605 linear course path

CS605 全教材线性学习：从自动机到 NP-complete

这页按 annotated notes 的时间顺序学习。先学什么是字符串和机器，再学语言类别，最后学不可判定性、归约和复杂度。每一步都先解释为什么需要这个概念。

1. 数学和语言地基

理论计算机的对象不是某个 Java 程序，而是“字符串集合”。先学 alphabet、string、language、set、function、proof by contradiction，后面所有证明才有落脚点。

顺序内容过关标准材料
1.1Mathematical preliminaries能读懂集合、函数、量词、证明结构。2026-02-03-Notes-Mathematical_preliminaries_annotated.pdf, CS605 - Mathematical preliminaries.pdf
1.2Historical foundations知道课程关心“什么能被计算”。A - Historical Foundations.pdf, CS605 Overview.pdf

2. Finite Automata 和 Regular Languages

Finite automaton 像一个只有有限记忆的机器。Regular language 就是这种机器能识别的语言。它能处理简单模式，但不能无限计数。

顺序内容过关标准材料
2.1Finite automata能画状态图，知道 start/accepting states。2026-02-03-Notes-FAs_annotated.pdf, 2026-02-04-Notes-FAs_cont_annotated.pdf
2.2Regular languages知道 finite languages regular，能理解非正则证明为什么需要 pumping lemma。2026-02-04-Notes-Regular_languages_annotated.pdf

3. PDA 和 Context-Free Languages

PDA 比 FA 多一个 stack，所以能处理一类嵌套或配对结构。Context-free language 是 PDA 能识别的语言。

顺序内容过关标准材料
3.1PDA能解释 stack 为什么能处理 a^n b^n。2026-02-05-Notes-PDAs_annotated.pdf
3.2CFL知道 not CFL proof 为什么需要考虑 uvxyz 的位置。2026-02-06-Notes-Context_free_languages_annotated.pdf

4. Countability、Church-Turing 和 Turing Machines

这一段把“机器”从 FA/PDA 推到通用计算。Turing machine 是抽象计算机；Church-Turing thesis 说所有合理计算模型本质上能计算同一类函数。

顺序内容过关标准材料
4.1Countable / uncountable理解为什么“语言比机器多得多”。2026-02-11-Notes-Countable_and_uncountable_annotated.pdf
4.2Church-Turing thesis知道 TM 为什么是计算能力的标准模型。2026-02-18-Notes-Church-Turing-thesis_annotated.pdf
4.3T-r and decidable能区分 recogniser 和 decider。2026-02-18-Notes-T-r-and-decidable_annotated.pdf, B - Computability I.pdf

5. Decidability 和 Undecidability

Decidable 意味着一定停机并给 yes/no。Undecidable 意味着不存在这样的通用算法。HALT 是核心已知难题。

顺序内容过关标准材料
5.1Decidable examples能构造 TM pseudocode 证明语言 decidable。2026-02-25-Notes-Decidable_annotated.pdf
5.2Non-T-r languages理解 complement、co-T-r、不可识别。2026-02-25-Notes-Non-T-r-languages_annotated.pdf
5.3Mapping reductions会用 HALT 翻译到新问题。2026-03-04-Notes-Reductions_annotated.pdf, 2026-03-11 Reductions with non-T-r languages_annotated.pdf, reduction videos

6. Complexity：从能不能算到算得快不快

Computability 问“有没有算法”。Complexity 问“算法要多久”。Polynomial time 是可扩展计算的基本界线。

顺序内容过关标准材料
6.1Asymptotic analysis能比较 n, n^2, 2^n。2026-03-25-Notes-CCT_asymptotic_analysis_annotated.pdf, Growth rates.pdf
6.2P and PTIME知道 P 是 polynomial-time solvable。2026-03-25-Notes-CCT_PTIME_annotated.pdf
6.3NP知道 certificate 和 verifier。2026-04-01-Notes-CCT_NP_annotated.pdf
6.4NP-complete知道 in NP + NP-hard 两步证明。NPc1.pdf, NPc2.pdf, NP videos

7. 最后接到考试

读完教材后再回到 CS605 题型页，按 Q1-Q7 模板练 sample A/B。教材给理解，题型页给考试写法。
